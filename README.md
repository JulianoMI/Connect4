## Installation & Setup

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/JulianoMI/Connect4.git
   cd TwoPlayer
   ```

2. **Install Python dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**
   ```bash
   python main.py
   ```

4. **Open your browser**
   Navigate to `http://localhost:8000`
